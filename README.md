## Brisq

A simple fast raycast focus extension for linux built with tauri and rust.

## Download

You can download the latest release from AUR by the package manager of your choice using

```
yay -S brisq
```

## Development

Contributions are welcome! Please follow the guidelines below:

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them.
4. Push your changes to your fork.
5. Submit a pull request.

```
pnpm install
pnpm tauri:dev
```
